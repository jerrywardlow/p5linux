Using JSON

JSON endpoints can be accessed using the following syntax:

List of all categories: localhost:5000/categories/JSON/

List of all items: localhost:5000/items/JSON/

Information about a speicifc category: localhost:5000/category/'Category ID'/JSON/

Information about a specific item: localhost:5000/item/'Item ID'/JSON

Information about all items in a specific category: localhost:5000/category/'Category ID'/items/JSON/
