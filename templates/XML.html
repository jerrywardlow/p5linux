Using XML

XML endpoints can be accessed using the following syntax:

List of all categories: localhost:5000/categories/XML/

List of all items: localhost:5000/items/XML/

Information about a speicifc category: localhost:5000/category/'Category ID'/XML/

Information about a specific item: localhost:5000/item/'Item ID'/XML

Information about all items in a specific category: localhost:5000/category/'Category ID'/items/XML/
